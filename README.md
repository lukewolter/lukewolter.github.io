### The Rock Lore Vault
A personal AI-powered project exploring the wild, wacky histories behind iconic rock songs. This demonstrates my proficiency in using AI tools for content creation, web development, and analytics-driven strategy.

- **Overview**: Built exclusively with AI tools to create business (content strategy), tech (web app), and media (videos, podcasts) assets. The platform includes a responsive website, YouTube channel, and Spotify podcast.
- **Key Achievements**:
  - Orchestrated AI workflows with n8n, OpenAI GPT-4, Grok, DALL-E, and pictory.ai for script generation, image creation, and video/podcast production.
  - Developed a MERN-stack website using devin.ai, hosted on AWS EC2 Lightsail, with integrations for YouTube and Spotify APIs.
  - Conducted market research via Google Analytics, Looker Studio, and Amplitude, achieving early engagement metrics like 50%+ video watch time.
- **Skills Demonstrated**: Product roadmap planning, AI automation, data-driven decision-making, full-stack development, SEO optimization, and multimedia content strategy.
